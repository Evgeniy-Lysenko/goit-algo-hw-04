# Домашнє завдання №4  
## Консольний бот-помічник
Файл - homework4_m4.py
У цьому домашньому завданні реалізовано простого консольного бота-помічника, який розпізнає текстові команди, введені користувачем, та виконує відповідні дії.  
Бот працює в інтерактивному режимі та зберігає контакти у словнику Python.

---

## Опис функціональності

Бот підтримує такі команди:

- **hello**  
  Виводить привітальне повідомлення.

- **add \<ім'я> \<номер>**  
  Додає новий контакт до словника.

- **change \<ім'я> \<новий номер>**  
  Оновлює номер телефону для існуючого контакту.

- **phone \<ім'я>**  
  Виводить номер телефону за вказаним ім’ям.

- **all**  
  Виводить усі збережені контакти та їхні номери телефонів.

- **close / exit**  
  Завершує роботу бота.

Усі команди розпізнаються незалежно від регістру введення.

---

## Структура програми

- `main()` — основний цикл обробки команд користувача  
- `parse_input()` — розбір введеного рядка на команду та аргументи  
- `add_contact()` — додавання контакту  
- `change_contact()` — зміна номера телефону  
- `contact_phone()` — пошук номера телефону за ім’ям  
- `all_contacts()` — виведення всіх збережених контактів  

Для зберігання контактів використовується словник, де:
- ключ — ім’я контакту
- значення — номер телефону
